Step 1 : Download

1. Open the official website. (Click here.)
2. Click on yellow box for download the lastest python installer and wait a second to download installer.


Step 2 : Installation

1. Run The Python Installer
2. Choose Upgrade Now to keep yor current settings
or choose Customize installation to enable or disable features.

(If you haven't installed python, it will show Install Now on Upgrade Now .)


3. Wait a second to install.


4. Setup was sucessful, Press to close.
