Python-Installation

Step 1 : Download

1. Open the official website.
2. Click on yellow box for download the lastest python installer and wait a second to download installer.


Step 2 : Installation

1. Run The Python Installer
2. Choose Upgrade Now to keep yor current settings
or choose Customize to enable or disable features.


3. Wait a second to install.


4. Setup was sucessful, Press to close.


Step 3 : Verify the python installation

You can verify if the Python installation is successful either through the command line
